# 객관식

 1. 식사 기록 및 AI 식습관 분석
 - 급식은 학교 정보를 연동하여 메뉴를 자동으로 기록하고, 그 외 음식은 사용자가 카메라로 찍어 등록한다. 단, 제대로 등록이 안될 경우에는 수기 입력을 한다
 - 식사 시간과 섭취한 메뉴를 지속적으로 기록·분석하여 식습관 패턴을 관리한다.
 반복되는 식습관을 기반으로 생활습관 피드백(그림 + 세부 글)을 제공한다.
 예) "이번 주에는 매운 음식 섭취가 많아 소화불량이 발생할 가능성이 있습니다."
 예) "당분이 많은 음식을 자주 섭취하여 피부 트러블이 나타날 수 있습니다."


 2. 개인 건강 상태 기반 맞춤 가이드
 - 사용자가 입력한 알레르기, 식습관(채식 등), 건강 상태를 바탕으로 음식별 주의사항을 안내한다.
(음식별 주의사항 안내하면서 메뉴를 추천해줌)
 - 특정 음식을 제한하기보다, 섭취 후 발생할 수 있는 증상과 이를 보완할 수 있는 음식이나 식습관을 함께 추천한다.
 예) "매운 음식을 섭취했다면 위장 부담을 줄이기 위해 충분한 수분을 섭취하거나 위에 부담이 적은 음식을 함께 섭취하는 것을 권장합니다."

 3. 메뉴별 식사 순서 가이드
 - 급식 메뉴를 분석하여 권장 섭취 순서를 안내한다.
 예) 채소 → 단백질 → 탄수화물 순으로 섭취하도록 제안하여 균형 잡힌 식사를 돕는다.
